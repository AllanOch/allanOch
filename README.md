# Donkey Kong-inspirert spill

Et enkelt 2D-spill inspirert av Donkey Kong. Spilleren MaTs må klatre plattformer og unngå ølbokser som kastes av Christian.

## Slik spiller du

- Bruk piltastene for å bevege deg.
- Hopp med opp-pilen.
- Unngå ølbokser som kastes fra toppen.
- Hvis du blir truffet av en ølboks, starter spillet på nytt.

## Filene

- `index.html` - spillrammeverk og oppsett.
- `style.css` - stil og layout.
- `script.js` - spilllogikk, bevegelser og kollisjoner.

## Kjør spillet

Åpne `spill3/index.html` i en nettleser for å spille.
